Arc

A practical learning sequence

Repeat the workflow until it becomes instinct: read, assess, verify, conclude, and communicate.

1. Read

Start with the requirement, the standard, and the exact facts on the page. No wandering before the scene is clear.

2. Assess

Ask what matters, what is missing, and where the risk or judgment point actually lives.

3. Verify

Test the answer against evidence, rules, and internal consistency before you trust the conclusion.

4. Communicate

State the result with clean language so the next person can use it without decoding a fog machine.
